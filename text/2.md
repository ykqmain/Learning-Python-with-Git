## 语句

**注意：缩进，冒号**

### 运算符

1. 逻辑运算
    * `and`  且 (短路运算)
    * `or`  或
    * `not`  非
<br>

2. 其它
    * `in`，`not in` (成员运算符)
    * `is`，`is not` (身份运算符)
<br>

3. pass

<br><br>


### 条件

```
if 条件:
    语句
elif 条件:
    语句
else:
    语句
```

<br><br>


### 循环

```
for ... in ...:
    语句
```

* range(start, stop, step)
* else

<br><br>

```
while 条件:
    语句
```

* else

<br><br>

* `break`

* `continue`

<br><br>

### pass

空语句，是为了保持程序结构的完整性。

pass 不做任何事情，一般用做占位语句。
